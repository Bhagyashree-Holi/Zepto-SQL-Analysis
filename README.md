# Zepto SQL Analysis

## Project Overview

This project analyzes Zepto product data using SQL to extract business insights related to pricing, discounts, product categories, and inventory.

## Dataset

- zepto_v2.csv

## Tools Used

- SQL
- CSV Dataset
- GitHub

## Analysis Performed

- Data Cleaning
- Product Analysis
- Category-wise Analysis
- Discount Analysis
- Pricing Insights
- Business Queries using SQL

## Files

- zepto_sql_proj.sql
- zepto_v2.csv

## Key Objectives

- Understand product pricing patterns
- Analyze discount distribution
- Identify top-performing categories
- Generate actionable business insights using SQL
